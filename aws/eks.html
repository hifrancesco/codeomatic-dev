Amazon EKS Curriculum:

Beginner EKS Curriculum:

  1. Step 1: Introduction to EKS

    • Understanding container orchestration and Kubernetes
    • Introduction to Amazon EKS and its features
    • Creating an EKS cluster using AWS Management Console
    • Interacting with EKS cluster using kubectl

  2. Step 2: Deploying Applications on EKS

    • Creating and managing EKS worker nodes
    • Deploying applications using EKS and Kubernetes manifests
    • Scaling applications with EKS Autoscaling
    • Managing application configurations with ConfigMaps and Secrets

  3. Step 3: Networking and Load Balancing in EKS

    • Understanding networking in EKS clusters
    • Configuring and managing network policies
    • Using AWS Load Balancer Controller
    • Ingress controllers and application routing

  4. Step 4: Managing Storage on EKS

    • Understanding storage options in EKS
    • Configuring and using Amazon EBS volumes
    • Dynamic provisioning with Amazon EFS
    • Stateful applications and PersistentVolumes

Intermediate EKS Curriculum:

  1. Step 5: Advanced Deployment Techniques

    • Managing application releases with Helm
    • Blue-green and canary deployments on EKS
    • Using AWS App Mesh for advanced service mesh features
    • Application lifecycle management and versioning

  2. Step 6: Observability and Monitoring in EKS

    • Monitoring EKS clusters with CloudWatch and Prometheus
    • Collecting and analyzing logs with CloudWatch Logs and Elasticsearch
    • Visualizing metrics and logs with Grafana
    • Setting up alerts and notifications

  3. Step 7: Autoscaling and High Availability

    • Scaling EKS clusters with Cluster Autoscaler
    • Horizontal Pod Autoscaling and Vertical Pod Autoscaling
    • Ensuring high availability with multiple Availability Zones
    • Implementing pod anti-affinity and node taints/tolerations

    4. Step 8: Security and Governance

    • Securing EKS clusters with IAM roles and policies
    • Implementing network security with VPC and Security Groups
    • Enforcing access controls and RBAC in EKS
    • Monitoring and auditing EKS clusters for compliance

Advanced EKS Curriculum:

  1. Step 9: Advanced Networking in EKS

    • Implementing advanced networking configurations in EKS
    • Using AWS PrivateLink for secure communication
    • Integrating with external network services and VPC peering
    • Securing network traffic with mTLS and encryption

  2. Step 10: Advanced Cluster Operations

    • Upgrading EKS clusters to newer Kubernetes versions
    • Implementing cluster autoscaling policies
    • Managing cluster add-ons and customizing the control plane
    • Debugging and troubleshooting common EKS issues

  3. Step 11: CI/CD with EKS

    • Setting up CI/CD pipelines for EKS applications
    • Using AWS CodePipeline, CodeBuild, and CodeDeploy
    • Implementing GitOps workflows with EKS
    • Infrastructure as Code using AWS CloudFormation or Terraform

  4. Step 12: Advanced EKS Architecture

    • Designing highly available and scalable EKS architectures
    • Using EKS with AWS services like AWS Fargate, AWS Lambda
    • Implementing multi-region and multi-account EKS setups
    • Optimizing EKS performance and resource utilization
